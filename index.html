1. Tecnologías Web

¿Qué es la Web?

WEB (World Wide Web, o www), es un conjunto de documentos (webs) interconectados por enlaces de hipertexto, disponibles en Internet que se pueden comunicar a través de la tecnología digital. Se entiende por hipertexto la mezcla de textos, gráficos y archivos de todo tipo, en un mismo documento. (Latorre,2018)

Web e Internet no son sinónimos; Internet es la red de redes donde reside toda la información, siendo un entorno de aprendizaje abierto, más allá de las instituciones educativas formales. La web es un subconjunto de Internet que contiene información a la que se puede acceder usando un navegador.

Tanto el correo electrónico, como Facebook, Twiter, TikTok, blogs, juegos, tiendas, etc. son parte de Internet, pero no la web.

Tecnologías Web

Las tecnologías Web sirven para acceder a los recursos de conocimiento disponibles en Internet o en las intranets utilizando un navegador.

✔ Ventajas

La principal ventaja de la tecnología web es que ofrece comodidad y una alta velocidad de comunicación en el mundo informático.

Ya sea en la oficina o en el hogar, los procesos que utilizan una computadora son más rápidos y sencillos con el uso de una red.

La tecnología web permite que los mensajes se envíen a través de un sistema, mientras que antes podría haber sido necesario emplear un corredor o abandonar su espacio de trabajo para comunicar un mensaje.

Es claro ver cómo la tecnología web reduce costos y hace que una empresa sea más eficiente, elevando el potencial comercial.

✗ Desventajas

Los asuntos relacionados con la tecnología web pueden ser muy complicados y sería difícil para alguien sin experiencia relevante resolver un problema de red.

Esto significa que es necesario contratar a alguien con las habilidades específicas para resolver problemas de red, lo que cuesta dinero.

La existencia de una red brinda la oportunidad de un ataque al sistema informático. Se podrían aprovechar las debilidades de una red; la información importante podría ser robada o destruida y el malware podría infectar los diversos sistemas de red.

La seguridad de la red es otro tema que debe tenerse en cuenta al utilizar la tecnología web.

Protocolo HTTP

HiperText Transfer Protocol (HTTP) es el protocolo para la transferencia de archivos en la web que maneja la interacción entre un navegador (cliente) y un servidor web.

Servidores Web

Son programas que permite gestionar aplicaciones en el lado del servidor, estableciendo conexiones en ambas direcciones

El cliente mediante internet, realiza una petición al servidor web, éste busca en sus repositorios y devuelve la página solicitada, el formato de los datos que transmiten es en forma de páginas web, hipertextos o páginas HTML; textos complejos con enlaces, figuras, formularios, botones y objetos incrustados como animaciones o reproductores de sonido
